# Space Environment Data Analysis

## Overview
This project analyzes space environment datasets such as radiation levels, solar activity, and atmospheric conditions.

## Objectives
- Analyze space environment datasets
- Perform statistical analysis
- Visualize space weather patterns

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- SQL

## Project Structure
data/ – raw datasets  
notebooks/ – Jupyter notebooks  
scripts/ – analysis scripts
